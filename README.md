# NECOMAtter

This is NECOMAtter server.

## How to use

1. Install python and neo4j
2. Install following python packages:
 - gevent
 - flask
 - py2neo
 - requests
 - dateutil
3. Run index.py (You can use NECOMAtter_service.sh)
4. Add initial user. use tools/user_add.py
